# 📝 To-Do List Console App 

Um sistema interativo de lista de tarefas desenvolvido em **Java 21**, focado na aplicação prática de Programação Orientada a Objetos e manipulação de coleções dinâmicas. O projeto utiliza um menu via console para gerenciar o ciclo de vida de atividades.

## 🚀 Funcionalidades

* **Menu Interativo:** Interface via CLI (Command Line Interface) com opções numeradas.
* **Adição Dinâmica:** Permite criar novas tarefas instanciando objetos da classe `Activities`.
* **Listagem com Status:** Exibe as tarefas cadastradas mostrando quais já foram concluídas utilizando marcações visuais (ex: `[X]` para concluído).
* **Gestão de Conclusão:** Permite alterar o status de uma tarefa específica através do seu índice no array.

## 🧠 Detalhes Técnicos e Lógica

O projeto foi estruturado para demonstrar competências em:

* **POO (Programação Orientada a Objetos):** Separação de responsabilidades entre a classe principal (`Main`) e a classe de modelo (`Activities`).
* **ArrayList & Generics:** Uso de `ArrayList<Activities>` para armazenamento dinâmico, superando as limitações de tamanho fixo dos arrays convencionais.
* **Estruturas de Repetição (For):** Iteração sobre a lista para exibição e busca de dados.
* **Estruturas Condicionais (Switch/Case & If/Else):** Controle de fluxo do menu e validação de índices para evitar erros de execução (*IndexOutOfBounds*).
* **Interatividade:** Uso da classe `Scanner` para entrada de dados em tempo real.

## 🛠️ Tecnologias e Ambiente

* **Linguagem:** Java (JDK 21)
* **IDE:** Apache NetBeans
* **Paradigma:** Orientação a Objetos
